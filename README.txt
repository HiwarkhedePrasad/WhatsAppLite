WhatsApp Lite - Lightweight Desktop Client
==========================================

Version: 1.0.0
Author: [Your Name/Antigravity]

INSTALLATION
------------
This application is a portable, single-file executable. No complex installation is required.

1.  Copy the `WhatsAppLite.exe` file to any location on your computer (e.g., Desktop, Documents).
2.  Double-click `WhatsAppLite.exe` to run.

PREREQUISITES
-------------
To run this application, you must have the following installed:

1.  .NET Desktop Runtime 8.0.x (x64)
    Download here: https://dotnet.microsoft.com/en-us/download/dotnet/8.0
    (Look for ".NET Desktop Runtime" under Windows x64)

2.  WebView2 Runtime
    (Usually pre-installed on Windows 10/11. If missing, the app may prompt or crash.)
    Download here: https://developer.microsoft.com/en-us/microsoft-edge/webview2/

FEATURES
--------
- Lightweight (~120MB RAM usage)
- Persistent login (scan QR code once)
- Native Windows behavior (Snap, Maximize, Aero Shake)
- Custom dark title bar with standard window controls
- Minimized to taskbar
- Instant startup

TROUBLESHOOTING
---------------
- If the app doesn't open, ensure you have installed the .NET Desktop Runtime 8.0.
- If the screen is blank, check your internet connection or install WebView2 Runtime.
